Convert CSV to a newline-delimited JSON file, where each row is mapped to an object with the column names as keys.

Example

Input

NameTDIntYards
Tom Brady21466
Matt Ryan20284

Output

to_json("twentyeight-three.ndjson")

{"Name":"Tom Brady","TD":2,"Int":1,"Yards":466}

{"Name":"Matt Ryan","TD":2,"Int":0,"Yards":284}

5 ## Motivation
6 There's plenty of other CSV parsers in the wild, but I had a hard time finding what I wanted. Specifically, I wanted something which had an interface similar to Python's `csv` module. Furthermore, I wanted support for special use cases such as calculating statistics on very large files. Thus, this library was created with these following goals in mind:
7 
8 ### Performance
9 This CSV parser uses multiple threads to simulatenously pull data from disk and parse it. Furthermore, it is capable of incremental streaming (parsing larger than RAM files), and quickly parsing data types.
10 
11 ### RFC 4180 Compliance
12 This CSV parser is much more than a fancy string splitter, and follows every guideline from [RFC 4180](https://www.rfc-editor.org/rfc/rfc4180.txt). On the other hand, it is also robust and capable of handling deviances from the standard. An optional strict parsing mode can be enabled to sniff out errors in files.
13 
14 ### Easy to Use and [Well-Documented](https://vincentlaucsb.github.io/csv-parser)
15 
16 In additon to being easy on your computer's hardware, this library is also easy on you--the developer. Some helpful features include:
17  * Decent ability to guess the dialect of a file (CSV, tab-delimited, etc.)
18  * Ability to handle common deviations from the CSV standard, such as inconsistent row lengths, and leading comments
19  * Ability to manually set the delimiter and quoting character of the parser

23 ## Building
24 All of this library's essentials are located under `src/`, with no dependencies aside from the STL. This is a C++17 library developed using Microsoft Visual Studio and compatible with g++ and clang. The CMakeList and Makefile contain instructions for building the main library, some sample programs, and the test suite.

26 **GCC/Clang Compiler Flags**: `-pthread -O3 -std=c++17`

42 ## Features & Examples
43 ### Reading a Large File
44 With this library, you can easily stream over a large file without reading its entirety into memory.
45 
46 ```cpp

48 
49 using namespace csv;
50 
51 ...
52 
53 CSVReader reader("very_big_file.csv");
54 std::vector<std::string> row;
55 
56 while (reader.read_row(row)) {
57  // Do stuff with row here
58 }
59 
60 ```

62 ### Indexing Rows by Column Name and Type Conversions
63 If your CSV has lots of numeric values, you can also have this parser automatically convert them to the proper data type.
64 For efficiency, numeric values are lazily converted, and the indexing feature is implemented by having all rows
65 share a pointer to the original set of column names.
66 
67 ```cpp
68 # include "csv_parser.hpp"
69 
70 using namespace csv;
71 
72 ...
73 
74 CSVReader reader("very_big_file.csv");
75 std::vector<CSVField> row;
76 
77 while (reader.read_row(row)) {
78  if (row["timestamp"].is_int())
79  row["timestamp"].get<int>();
80 
81  // get<>() returns a std::string_view of the original field
82  for (size_t i = 0; i < row.size(); i++) {
83  std::cout << row[i].get<>() << ...
84  }
85 }
86 
87 ```

89 ### Specifying a Specific Delimiter, Quoting Character, etc.
90 Although the CSV parser has a decent guessing mechanism, in some cases it is preferrable to specify the exact parameters of a file.
91 
92 ```cpp
93 # include "csv_parser.hpp"
94 # include ...
95 
96 using namespace csv;
97 
98 CSVFormat format = {
99  '\t', // Delimiter
100  '~', // Quote-character
101  '2', // Line number of header
102  {} // Column names -- if empty, then filled by reading header row
103 };
104 
105 CSVReader reader("wierd_csv_dialect.csv", {}, format);
106 vector<CSVField> row;
107 
108 while (reader.read_row(row)) {
109  // Do stuff with rows here
110 }
111 
112 ```
